function intercept(this,LAI,snowfall,rainfall,windCanopy)

writen by Shen, Xinyi, April, 2015

contact: xinyi.shen@uconn.edu

Algorithm Description

calculate the mass(snow/rainfall) flux caused by canopy interception and update the storage accordingly this function should be called before solving the energy balances at every time step

output arguments

intSnow (mm) intercepted snow updated by the snowfall in the current time step intRain (mm) intercepted rain updated by the

input arguments

LAI: leaf area index

snowfall (mm): accumulated snow fall during the current time step, i.e., total snow fall

rainfall (mm): accumulated snow fall during the current time step, i.e., total rainfall

windCanopy(m/s): adjusted windspeed at the canopy height

main routine

global LAI_SNOW_MULTIPLIER MIN_SWQ_EB_THRES

Determine the maximum snow interception water equivalent

Kobayashi, D., 1986, Snow Accumulation on a Narrow Board, Cold Regions Science and Technology, (13), pp. 239-245. Figure 4.

Imax1 = 4.0* LAI_SNOW_MULTIPLIER * LAI;
isCold=this.Tfoliage<=-3;
maxSnowInt=4*(this.Tfoliage>=-1)+(isCold)+(this.Tfoliage*1.5+5.5).*((this.Tfoliage<1)&(~isCold));
maxSnowInt=maxSnowInt.*LAI_SNOW_MULTIPLIER * LAI;

Calculate snow interception

dSnowInt=snowfall.*(1-this.intSnow./maxSnowInt);
dSnowInt=min(maxSnowInt-this.intSnow,dSnowInt);
dSnowInt(maxSnowInt<=0)=0;

Reduce the amount of intercepted snow if windy and cold.

Ringyo Shikenjo Tokyo, #54, 1952. Bulletin of the Govt. Forest Exp. Station, Govt. Forest Exp. Station, Meguro, Tokyo, Japan. FORSTX 634.9072 R475r #54. Page 146, Figure 10. Reduce the amount of intercepted snow if snowing, windy, and cold (< -3 to -5 C). Schmidt and Troendle 1992 western snow conference paper.

snowBlown=(DeltaSnowInt>0)& (dSnowInt>0)& isCold;
blownSnow = (0.2 * windCanopy(snowBlown) - 0.2).* dSnowInt(snowBlown);
blownSnow(blowSnow>dSnowInt(snowBlown))=dSnowInt(snowBlown);
dSnowInt(snowBlown)=dSnowInt(snowBlown)-blownSnow;
% if (*IntSnow +  DeltaSnowInt > Imax1) DeltaSnowInt =0.0; I do not see the necessity
this.snowThru=snowfall-dSnowInt;

isThin=(this.intSnow<MIN_SWQ_EB_THRES)&(snowfall==0);
this.snowThru(isThin)=this.snowThru(isThin)+this.intSnow(isThin);
this.intSnow(isThin)=0;

intSnow=this.intSnow+dSnowInt;
intSnow(intSnow<SMALL)=0;

capacity of rain intercepted on branches and stored in intercepted snow

maxWaterInt = LIQUID_WATER_CAPACITY*intSnow + this.Wim;
intRain=this.intRain+rainfall;
this.rainThru=max(0,intRain-maxWaterInt);
this.intRain=this.intRain-this.rainThru;

isThin=(this.intRain<MIN_SWQ_EB_THRES)&(rainfall==0);
this.rainThru(isThin)=this.rainThru(isThin)+this.intRain(isThin);
this.intRain(isThin)=0;

triger unloading if overload

totalWater=this.intRain+this.intSnow;
isOverload=totalWater>Imax1;
overload = this.intSnow(isOverload) + this.intRain(isOverload) - Imax1(isOverload);
intRainFract= this.intRain(isOverload)./totalWater(isOverload);
unloadRain=overload.*intRainFract;
unloadSnow=overload.*(1-intRainFract);
this.intRain(isOverload) = this.intRain(isOverload) - unloadRain;
this.intSnow(isOverload) = this.intSnow(isOverload) - unloadSnow;
this.rainThru(isOverload) = this.rainThru(isOverload) + unloadRain;
this.snowThru(isOverload) = this.snowThru(isOverload) + unloadSnow;

end
